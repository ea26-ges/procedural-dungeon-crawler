# Project Overview

This project is a procedural dungeon crawler game that utilizes procedural generation techniques to create unique dungeons for each play session. The game aims to provide an engaging experience with diverse environments, challenges, and enemies.

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ea26-ges/procedural-dungeon-crawler.git
   cd procedural-dungeon-crawler
   ```

2. **Install dependencies**:
   Make sure you have the necessary dependencies installed. Run:
   ```bash
   npm install
   ```

3. **Run the game**:
   Once the dependencies are installed, you can start the game with:
   ```bash
   npm start
   ```

4. **Enjoy!**
   Explore the procedurally generated dungeons and have fun!
